## How to build üèóÔ∏è

To build the project you have to download devkitPro, install devkitPro and follow these commands :

``sudo pacman -S switch-curl switch-zlib``  
``git clone https://github.com/PoloNX/AtmoPackUpdater``  
``cd AtmoPackUpdater``  
``make``  
``cd amssu-rcm``
``make``
